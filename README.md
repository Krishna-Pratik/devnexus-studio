# Devnexus Studios

## About

This repository contains the Devnexus Studios website built with React and Vite.

## Local Development

1. Install dependencies:

```bash
npm install
```

2. Start the development server:

```bash
npm run dev
```

3. Build frontend for production:

```bash
npm run build
```

4. Start backend in production mode:

```bash
npm run start
```

5. Preview frontend production build locally:

```bash
npm run preview
```
